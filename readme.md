# Chat Bot API

A simple API for building and integrating chat bots.

## Features

- Easy integration
- RESTful endpoints
- Scalable architecture

## Getting Started

1. Clone the repository.
2. Install dependencies:  
    ```bash
    npm install
    ```
3. Start the server:  
    ```bash
    npm start
    ```

## API Endpoints

| Method | Endpoint      | Description         |
|--------|--------------|---------------------|
| POST   | `/chat`      | Send a message      |
| GET    | `/messages`  | List all messages   |
