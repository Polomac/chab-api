# Chat Bot API

A simple API for building and integrating chat bots.

## Features

- Easy integration
- RESTful endpoints
- Scalable architecture

## Getting Started

1. Clone the repository.
2. Install dependencies:  
    ```bash
    npm install
    ```
3. Run locally:  
    ```bash
    npm run dev
    ```
4. Build:  
    ```bash
    npm run build
    ```
5. Start the server:  
    ```bash
    npm run start
    ```
